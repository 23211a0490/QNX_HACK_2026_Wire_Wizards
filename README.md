# QNX_HACK_2026_Wire_Wizards

## 📌 Overview

This project is a **CPU simulation and embedded system application** developed as part of **QNX Hack 2026**.

It demonstrates:

* Low-level system programming
* Hardware interaction
* Modular design using C

---

## ⚙️ Features

* CPU simulation logic implemented in C
* OLED display integration
* Temperature simulation module
* Modular code structure
* Build system using Makefile

---

## 📂 Project Structure

```
.
├── src/
│   ├── cpu.c
│   ├── oled_display.c
│   └── temp_sim.c
├── Makefile
├── build/
└── cpu.core
```

---

## 🚀 Getting Started

### 🔧 Prerequisites

* C Compiler (GCC / QNX toolchain)
* Make

### ▶️ Build the project

```
make
```

### ▶️ Run the program

```
./cpu
```
